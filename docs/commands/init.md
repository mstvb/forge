# init

Initialize a new Forge repository in the current directory.

## Synopsis
```
forge init
```

## Description
Creates a `.forge` directory with `objects/`, `commits/`, an empty `HEAD`, and an empty `index`. If a repository already exists, a message is shown and nothing is changed.

## Examples
```
PS C:\path\to\project> forge init
[Forge] >> Repository erfolgreich initalisiert
```
